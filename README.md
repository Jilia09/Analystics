# User Data Generator & Analytics Platform

Цей проєкт призначений для автоматичної генерації синтетичних даних користувачів у форматі JSON та їх подальшого комплексного аналізу. За допомогою бібліотек **Pandas** та **Matplotlib** скрипт обробляє великі масиви даних, будує аналітичні метрики та візуалізує ключові бізнес-показники.

## 🚀 Функціонал проєкту

1. **Генерація даних**: Автоматичне створення 1000 унікальних профілів користувачів із випадковими, але реалістичними характеристиками (ім'я, країна, місто, навички, пристрої, тип підписки, зарплата, рейтинг тощо).
2. **Базовий аналіз**: Розрахунок основних статистичних показників (середній/медіанний вік, розподіл адміністраторів, активність користувачів).
3. **Групування та агрегація**: Аналіз заробітної плати, рейтингів та кількості користувачів у розрізі країн, міст та типів підписок.
4. **Пошук аномалій (IQR)**: Автоматичне виявлення викидів (outliers) у рівні заробітних плат за допомогою методу міжквартильного розмаху.
5. **Візуалізація даних**: Побудова графіків (лінійні тренди, стовпчасті та кругові діаграми, boxplot, scatter plots, а також теплова карта кореляцій).

## 🛠 Технологічний стек

* **Python 3.x** — основна мова розробки.
* **Pandas** — очищення, обробка та аналіз структурованих даних.
* **Matplotlib** — візуалізація даних та побудова графіків.
* **JSON & Random** — вбудовані модулі для генерації та збереження симуляційних даних.

## 📋 Перелік реалізованих аналітичних завдань

### 🔹 Базові метрики
* Підрахунок загальної кількості користувачів.
* Знаходження середнього, мінімального та максимального віку.
* Визначення кількості адміністраторів та відсотка активних акаунтів.
* Розрахунок середньої та медіанної зарплати.
* Виведення Топ-10 користувачів за рівнем доходу.

### 📊 Групування (Aggregation)
* Розподіл користувачів за країнами та містами.
* Середня зарплата у розрізі країн та типів підписки.
* Середній рейтинг користувачів за країнами.
* Середній вік залежно від типу пристрою.

### 🔍 Аналіз активності та фільтрація
* Пошук користувачів, які не заходили на платформу понад 180 днів.
* Фільтрація користувачів із високим рейтингом (> 4.5).
* Виявлення користувачів із нульовою кількістю проєктів або високою продуктивністю (> 500 виконаних завдань).
* Порівняння доходів активних користувачів та тих, чия зарплата вища за середню.

### 📅 Робота з датами
* Аналіз динаміки реєстрацій за місяцями та визначення пікового місяця.
* Розрахунок середнього "віку" облікового запису в днях.

### 📈 Візуалізація (Аналітичні графіки)
* **Лінійний графік**: Динаміка реєстрацій у часі.
* **Гістограми**: Розподіл віку та заробітної плати.
* **Boxplot**: Діаграма розмаху зарплат за країнами (для наочного аналізу медіани та квартилів).
* **Кругова діаграма**: Частка користувачів на кожному тарифному плані (`free`, `basic`, `premium`).
* **Scatter Plots**: Залежність зарплати від рейтингу та кількості проєктів.
* **Heatmap**: Теплова карта кореляції між усіма числовими ознаками проєкту.
